8.1 GraphQL

8.2 Was ist das

GraphQL is a query language for APIs.

What is a Query Language?

This is a programming language that requests and retrieves data from database- & information-systems by sending queries, similarly to SQL.

8.4 Typescript

8.4.1 Was ist das?

  • TypeScript adds additional syntax to JavaScript, in order to be able to find errors much more quickly in your Editor (for example, in VS-Code).
  • TypeScript code is convertible into JavaScript code. Thus, TypeScript is able to run anywhere, where JavaScript runs.

8.4.2 Quelle

  • Webseite (Home): https://www.typescriptlang.org/
  • Documentation: https://www.typescriptlang.org/docs/handbook/intro.html
